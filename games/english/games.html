Instructions

Element Match Game:

1. Click on identical element symbols to match and clear them

2. Or match related elements (such as elements from the same group)

3. Each matched group displays basic information about the element

4. The goal is to clear as many elements as possible before time runs out

5. Use the hint function if you need help

Chemistry Sheep Game:

1. Click on elements to place them in the slots below

2. When there are three identical or related elements in the slots, they will be cleared

3. You can only have a maximum of 7 elements in the slots

4. The goal is to clear all element blocks

5. Use power-ups to help you complete the game

Learning Features:

- The game will randomly display basic information about chemical elements

- After completing the game, you will see the chemistry knowledge you learned during gameplay

- You can adjust the difficulty and learning content in the game settings
