Game Instructions

Game Rules:

1. Click on identical element symbols to match and clear them

2. Or match related elements (such as elements from the same group)

3. Each matched group displays basic information about the element

4. The goal is to clear as many elements as possible in 60 seconds

5. Each match earns 10 points, with bonus points for consecutive matches

Learning Features:

- The game displays element symbols, names, atomic numbers, and other basic information

- After completing the game, you will see the chemistry knowledge you learned during gameplay

- You can deepen your understanding of the periodic table through gameplay
